Theme and UI are now an agent-readable contract.

This slice makes styling less ambiguous for coding agents. The generated app has a structured token source of truth, a typed wrapper imported by application code, an unstyled primitive registry, and validation that catches missing tokens, invalid token types, malformed token JSON, and mirror drift.

packages/theme/tokens.json @app/theme @app/ui agentstack theme validate

What changed

Theme tokens are structured

packages/theme/tokens.json carries colors, typography, spacing, radius, shadow, motion, and density. Core validates required paths and value types.

The typed wrapper cannot drift

packages/theme/src/index.ts imports ../tokens.json and exports themeTokens, ThemeTokens, and token role helpers.

UI primitives are unstyled

@app/ui exposes primitives for auth, org switching, plans, settings, tables, forms, modals, states, command search, lists, and navigation shells.

Validation surface

instead of duplicating token values.

and export themeTokens from that JSON source.
Blocks: validate, validate --cloud, deploy

Theme diagnostics run both through the focused theme validate command and the broader local gate used before cloud validation and preview deploy rehearsal.

Primitive model

PrimitiveWhy it existsAgent constraint
authGate, orgSwitcher, userMenu Common Clerk-auth surfaces should behave consistently across web and mobile. Use required states and token roles before adding surface-specific components.
planGate, settingsSection, dataTable, form B2B SaaS workflows need predictable loading, error, empty, selected, and disabled behavior. Render platform-native UI, but keep behavior and state names shared.
modal, commandSearch, mobileList, navigationShell Navigation and command surfaces are frequent sources of divergent ad hoc styling. Use colors.surface, colors.foreground, and colors.focusRing roles.

Why this matters for agents

Less guessing

Agents no longer need to infer which color, radius, or state names should be shared. The generated packages define stable vocabulary for product UI work.

More mechanical enforcement

A missing focusRing, string spacing value, malformed token JSON, or duplicated typed mirror produces a concrete diagnostic before deployment rehearsal.
