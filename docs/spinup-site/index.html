What we built, why it matters, and how the pieces fit.

Agentstack slice 1 is a local, executable proof of an agent-first framework contract. It does not call Convex, Clerk, Vercel, or EAS yet. It proves the command surface, validation model, generated project shape, cloud-drift loop, and telemetry inspection path that real adapters can later implement.

0 real provider API calls in this slice

The short version

What exists now

A local-first framework kernel with a manifest schema, generated app template, CLI command router, validation engine, filesystem-backed local-cloud adapter, JSONL telemetry store, and end-to-end prototype workflow test.

What it proves

An agent can bootstrap a project, validate local structure, initialize simulated provider state, reconcile drift, and inspect redacted telemetry without getting lost in provider dashboards, env state, or hand-written setup steps.
