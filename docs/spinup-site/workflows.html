Command journeys

The important outcome is not that individual functions exist. It is that an agent can move through a coherent workflow: create a project, inspect lifecycle state, add a typed feature, validate it, initialize local-cloud state, repair drift, rehearse a preview deploy locally, and inspect telemetry.

What each command proves

  1. create-agent-stack copies the template, replaces app tokens, and includes the local workspace-status vertical.
  2. add feature creates coordinated domain, Convex, surface, telemetry, and feature-doc anchors.
  3. add billing-plan creates coordinated billing-plan anchors across domain, Convex, web, mobile, telemetry, and docs.
  4. add event creates typed telemetry definitions and generated event docs.
  5. env set writes scoped local env state without exposing values in output.
  6. inspect summarizes app identity, generated anchors, enabled services, and preview local-cloud state.
  7. skills:inspect confirms the versioned repo-local skill pack and no-MCP guidance contract are present.
  8. validate checks manifest, env, telemetry policy, and generated files. Evidence: local-structure.
  9. validate:quality runs structural validation plus local package quality commands. Evidence: local-quality; scope: local filesystem and package commands only.
  10. env:inspect prints expected preview services and env bindings.
  11. preview:plan plans local-cloud repair before preview:apply.
  12. preview:validate confirms local-cloud state matches the manifest. Evidence: local-rehearsal; scope: local-cloud state only, no live provider reads.
  13. doctor runs local validation plus preview local-cloud checks before provider, env, build, sync, or deploy work.
  14. dev is a preflight for local product work; it points agents at workspace-status surfaces plus validation, env, and sync commands without mutating providers.
  15. preview:deploy prints PLAN deploy preview without writing a deployment artifact. Evidence: local-rehearsal.
  16. preview:deploy:apply prints APPLIED deploy preview, writes .agentstack/deployments/preview.json, and records agentstack.deploy.completed. Evidence: local-rehearsal.
  17. prod:prepare checks production release readiness and reports repair commands.
  18. prod:provision plans production local-cloud state; prod:provision:apply applies that local state. Evidence: local-rehearsal.
  19. prod:validate runs release validation, then prod:deploy plans production deploy.
  20. prod:deploy:apply applies the local production artifact only and requires explicit production confirmation through the script. No real provider APIs are called.
  21. mobile:build:preview prints PLAN mobile build preview without writing a build artifact. Evidence: local-rehearsal.
  22. mobile:build:preview:apply prints APPLIED mobile build preview, writes .agentstack/builds/mobile-preview.json, and records agentstack.mobile.build.completed. Evidence: local-rehearsal.
  23. observe timeline --journey billing confirms billing-plan generation telemetry, including agentstack.billing-plan.added.

Validation pipeline

Failure and repair signals

SituationSignalRepair
Feature file already exists Add-feature refuses an unsafe overwrite. Choose a new feature name or update the existing feature anchors intentionally.
Event file already exists Add-event refuses an unsafe overwrite. Choose a new event name or update the existing generated event file intentionally.
Billing-plan file already exists Add-billing-plan refuses an unsafe overwrite. Choose a new billing plan name or update the existing generated billing-plan anchors intentionally.
Generated billing-plan anchor missing FAIL template.anchor.missing Restore the missing billing-plan file or rerun the generator in a clean project.
Generated file missing FAIL template.anchor.missing Restore generated anchor or regenerate the project.
Required env value missing FAIL env.custom.missing Run agentstack env set --env preview --surface convex --name STRIPE_MODE --value sandbox.
Cloud not initialized FAIL cloud.service.missing Run pnpm run preview:plan, then pnpm run preview:apply.
Manifest disables a linked service FAIL cloud.service.stale Run agentstack sync --env preview --apply to unlink stale state.
Preview deploy rehearsal not applied No .agentstack/deployments/preview.json Run pnpm run preview:deploy:apply; this writes local state only and does not call real providers.
Mobile build rehearsal not applied No .agentstack/builds/mobile-preview.json Run pnpm run mobile:build:preview:apply; this writes local state only and does not submit an EAS build.
